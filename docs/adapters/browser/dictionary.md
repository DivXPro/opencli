# Dictionary

**Mode**: 🌐 Public · **Domain**: `api.dictionaryapi.dev`

Search the open dictionary to quickly fetch native definitions, part of speech contexts, and phonetic pronunciations directly in your IDE terminal.

## Commands

| Command | Description |
|---------|-------------|
| `toycli dictionary search` | Fetch the exact definition of a word |
| `toycli dictionary synonyms` | Find related synonyms for a word |
| `toycli dictionary examples` | Read real-world sentence usage examples |

## Usage Examples

```bash
# Look up a complex term
toycli dictionary search serendipity

# Discover phonetics
toycli dictionary search ephemeral
```

## Prerequisites

- No browser required — utilizes the fast, open JSON definitions API.
